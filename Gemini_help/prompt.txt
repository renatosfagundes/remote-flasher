== Component 5: GearSelector.qml ==

A gear indicator display. For automatic: P R N D. For manual: N R 1 2 3 4 5 6. The active gear is highlighted, others are dim.

Properties:
  property int gearValue: 7      // -1=R, 0=N, 1-6=fwd, 7=P, 8=D
  property bool isAutomatic: true
  property real s: 1.0

Visual: The active gear letter/number should be large and bright (white or accent color) with a subtle glow/underline. Inactive gears are small and dim. This should look like a real digital gear indicator in a car dashboard.

Test wrapper (TestGearSelector.qml):

  import QtQuick.Window 2.15
  Window {
      width: 500; height: 150; visible: true; color: "#0c0c14"

          NumberAnimation { target: parent; property: "tg"; from: 0; to: -1; duration: 1500 }
          NumberAnimation { target: parent; property: "tg"; from: -1; to: 7; duration: 1500 }
      }
  }
